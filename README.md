# LT_Project
***
> Language Defination : Weight Converter.
***
Here I developed scanner for language translator(compiler).
---
I used "kg" ,"gram" ,"pound" ,"stone" as operators and "how", "means", "many" as keywords.
Here I make lex(.L) ,yacc(.Y) and C++ files for scanner.
